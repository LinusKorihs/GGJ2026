# GGJ2026

##How to play
###Controls
Move with WASD / Arrow Keys / Left Stick
Press Space / Controller South Button to Interact / Shapeshift
Press E / Controller North Button to Kill your target

###
You are tasked to kill one of the VIP's. 
Steal another guests appearance to not get spotted.
Find the agent to get a hint on how to identify your target.

But be careful, some rooms have restricted access and you need a certain appearance to enter.
